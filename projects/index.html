Gridworld & CartPole Player (ML)

Summer 2017
Tech Stack: Python (keras, numpy, scikit-learn), OpenAI Gym
This project implements and compares multiple Reinforcement Learning approaches to solve the Gridworld and CartPole problems.
• For Gridworld: MDP Value Iteration Algorithm and Q-Learning.
• For Cartpole: Deep Q-Network (DQN). Use of NN and DQN structures + mini-batch strategy + target network

Text Classification (ML)

Summer 2017
Tech Stack: Python (numpy, pandas, scikit-learn)
This project implements and compares two approaches to solve the Text Classification problem.
• 1st apporach: Naive Bayes with Laplace add-one Smoothing.

Sudoku CSP Solver (ML)

Summer 2017
Tech Stack: Java, JUnit
• This project implements a bot that solves sudoku boards using the CSP approach (Constraint Satisfaction Problem).
• Use of 3 heuristics: Most Constraining Value, Most Constrained Variable and Least Constraining Value

Project SIGECO (Web)

Fall 2015
Tech Stack: Java, Javascript, JSF, CSS, HTML, Glassfish Server and Oracle SQL
SIGECO is a web application for managing research projects convocations at University Icesi.
• The project covered all the stages of the software development life cycle (SDLC): Communication, Planning, Modeling, Construction, and Deployment.
• The application allows to manage the entire life cycle of a research project including: Proposal, Approval, Creation, Cancellatio, etc.
• SIGECO also allows to manage all the entities related with the research project
